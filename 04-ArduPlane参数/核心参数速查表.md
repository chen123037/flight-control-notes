## 3. 导航参数

### 3.1 L1 控制器

| 参数名 | 作用 | 默认值 | 调整建议 | 我的设置 |
|--------|------|--------|----------|----------|
| `NAVL1_PERIOD` | L1 周期 (s) | 15 | | |
| `NAVL1_DAMPING` | L1 阻尼 | 0.75 | | |

**L1 控制器原理**：
- L1 周期越小：转弯越急（跟踪精度高但可能振荡）
- L1 周期越大：转弯越缓（跟踪平滑但精度下降）

### 4.2 故障保护

| 参数名 | 作用 | 默认值 | 调整建议 | 我的设置 |
|--------|------|--------|----------|----------|
| `FS_SHORT_ACTN` | 短时失控动作 | 0 | | |
| `FS_LONG_ACTN` | 长时失控动作 | 0 | | |
| `FS_SHORT_TIMEOUT` | 短时失控超时 (s) | 1.5 | | |
| `FS_LONG_TIMEOUT` | 长时失控超时 (s) | 5 | | |

**失控动作选项**：
- 0 = 无动作
- 1 = RTL
- 2 = FBWA 滑翔
- 3 = 自动着陆
