# ArduPilot Plane 学习笔记

> 从零开始学习 ArduPilot 固定翼飞控系统

## 项目介绍

这是一个记录 ArduPilot Plane 学习过程的知识库，采用 AI 辅助学习机制，以文档为基础进行知识传递和积累。

## 学习目标

- 掌握 Mission Planner 地面站软件的使用
- 理解 ArduPilot 固定翼飞控参数
- 能够进行基本的飞行调参
- 了解 ArduPilot 源码架构（进阶）
