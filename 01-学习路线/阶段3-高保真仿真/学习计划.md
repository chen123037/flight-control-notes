## 关键技术点

### X-Plane 与 ArduPilot 连接架构

```
┌─────────────┐      UDP       ┌─────────────────┐
│   X-Plane   │ ←───────────→  │    ArduPilot    │
│  (气动仿真)  │  传感器数据     │   SITL (飞控)   │
│             │  控制指令       │                 │
└─────────────┘                └─────────────────┘
       ↑                              ↑
       │                              │
       └──────────┬───────────────────┘
                  │
           ┌──────┴──────┐
           │Mission Planner│
           │   (监控)     │
           └─────────────┘
```

### 16 翼控制分配思路

```
期望力/力矩 [Fx, Fy, Fz, Mx, My, Mz]^T
              │
              ▼
        控制分配矩阵 B
              │
              ▼
电机转速 [ω1, ω2, ..., ω16]^T

其中：τ = B * ω²
需要求解：ω = B⁺ * τ  (伪逆或优化)
```
