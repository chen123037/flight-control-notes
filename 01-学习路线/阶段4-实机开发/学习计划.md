## 安全须知

### 飞行测试安全原则

1. **人员安全**：所有人员与飞机保持安全距离（>20m）
2. **场地选择**：开阔场地，远离人群和建筑
3. **天气条件**：风速 < 5m/s，无雨
4. **应急准备**：随时准备切换到手动模式
5. **系留测试**：首飞前必须进行系留测试
6. **保险措施**：考虑购买无人机保险

### 飞行检查清单

```
飞行前检查：
□ 电池电压正常
□ 所有螺丝紧固
□ 螺旋桨无损伤
□ 传感器数据正常
□ 遥控器连接正常
□ 失控保护配置正确
□ GPS 定位良好（>8 颗星）
□ 天气条件适合
□ 场地安全
□ 应急计划准备
```

## 关键技术点

### 实机调参与仿真差异

| 项目 | 仿真 | 实机 |
|------|------|------|
| 气动模型 | 理想化 | 有误差 |
| 传感器噪声 | 模拟 | 真实 |
| 电机响应 | 理想化 | 有延迟 |
| 外部干扰 | 可控 | 不可控 |
| 试错成本 | 零 | 高（可能炸机）|

### 迭代调试流程

```
仿真参数 → 实机初测 → 数据分析 → 参数调整 → 再次测试
    ↑                                              │
    └──────────────────────────────────────────────┘
                      迭代优化
```
