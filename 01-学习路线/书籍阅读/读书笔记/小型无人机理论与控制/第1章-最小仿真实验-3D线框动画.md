# 第1章最小仿真实验：MAV 3D 线框动画（Python）

目标：用最简单的 Python 工具完成 MAV 线框模型，并实现旋转/平移动画。

## 环境要求

- Python 3.10+
- 依赖：`numpy`、`matplotlib`

安装：
```
pip install numpy matplotlib
```

## 运行方式

```
python chapter1_mav_wireframe.py
```

## 你会看到什么

- 一个接近书中三视图比例的 MAV 线框模型
- 默认是静态模型，可用鼠标拖拽旋转查看
- 需要动画时，把脚本里的 `ANIMATE = True`

## 可调参数

在脚本顶部的 `PARAMS` 中修改尺寸参数，例如机身长度、机翼宽度、尾翼高度。
