## 阶段目标

完成本阶段后，你应该能够：

1. ✅ 独立运行 ArduPilot SITL 仿真
2. ✅ 理解并能解释常用飞控参数的含义
3. ✅ 用 Python 实现简单的 PID 控制器
4. ✅ 理解坐标系变换和姿态表示
5. ✅ 阅读并理解 ArduPilot 部分源码
6. ✅ 具备基本的数学和物理直觉

## 阶段 1 综合考核

### 理论考核（AI 问答）

1. 解释什么是姿态，用数学方式表示
2. 画出飞控系统的基本控制回路框图
3. 推导 PID 控制器的离散化公式
4. 解释卡尔曼滤波的基本思想（预习）
5. ArduPilot 中 `PTCH2SRV_P` 参数的作用是什么

### 实践考核

1. **SITL 任务**：在 SITL 中完成一个包含起飞、航点、盘旋、降落的完整任务
2. **代码任务**：用 Python 实现一个 PID 控制器，控制一个模拟质点跟踪目标位置
3. **分析任务**：分析一段 SITL 飞行日志，指出哪些参数需要调整

### 考核标准

| 等级 | 标准 |
|------|------|
| 优秀 | 全部完成，理解深入，有自己的思考 |
| 良好 | 完成 80%，理解正确 |
| 及格 | 完成 60%，基本理解 |
| 需努力 | 完成不足 60%，需要补课 |
