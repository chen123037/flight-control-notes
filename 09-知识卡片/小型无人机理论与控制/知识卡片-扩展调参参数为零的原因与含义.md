### 主题：扩展调参参数为零的原因与含义

【一句话定义】  
扩展调参区的参数多为进阶控制与滤波功能，未启用或条件不足时会显示为 0 且不可调。

【核心概念】  
- 自稳 PID：控制姿态角  
- 速率 PID：控制角速度  
- 高度保持/航点导航：自主飞行功能  
- 滤波器：处理振动与噪声

【逐组说明】  
1) 自稳 Roll/Pitch/Yaw/悬停 PID  
- 作用：自稳模式下保持姿态稳定  
- P：比例响应  
- ACCEL MAX：姿态角加速度上限  
- 为 0 原因：未启用自稳模式、机型/固件不支持、传感器未校准  

2) Roll/Pitch/Yaw/悬停速率 PID + 基础滤波/油门参数  
- 作用：速率模式控制角速度  
- P/I/D/I 最大：角速度误差控制  
- FLTE/FLTD/FLTT：滤波与跟踪  
- Basic Filters：全局滤波  
- 油门加速度/油门速率：限制油门变化  
- 为 0 原因：速率模式未启用、陀螺仪未校准、硬件限制  

3) 高度保持/航点导航/滤波器（灰色不可调）  
- 高度保持：需气压计/激光雷达/GPS 等传感器  
- 航点导航：需 GPS 与任务航点  
- Filter Logs：滤波日志  
- Static Notch Filter：静态陷波  
- Harmonic Notch Filter：谐波陷波  
- 为 0 原因：功能未启用或未满足前置条件  

【判定逻辑】  
- 硬件条件：传感器是否接入且校准  
- 软件条件：模式是否启用  

【结论】  
参数为 0 且不可调，通常表示进阶功能未启用或条件未满足。
