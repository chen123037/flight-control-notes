### 主题：风干扰进入动力学的计算流程（图 4.9）

【一句话定义】
把稳定风与随机风合成为总风，再由空速推导攻角与侧滑角，使风扰动进入气动力与运动方程。

【适用场景】
风扰动建模、飞行动力学仿真链路搭建、控制鲁棒性分析。

【核心变量】
- `Vwb`：机体系总风速
- `Vab`：机体系空速
- `u,v,w`：机体系地速分量
- `α, β`：攻角与侧滑角
- `Rvb(ϕ,θ,ψ)`：惯性系到机体系的旋转矩阵

【计算流程（关键公式）】
1. 稳定风转换并合成随机风：
   - `Vwb = Rvb * Vw + Vwg`
2. 空速计算（地速减风速）：
   - `Vab = [u,v,w]^T - Vwb`
3. 空速与角度：
   - `Va = sqrt(ur^2 + vr^2 + wr^2)`
   - `α = arctan(wr / ur)`
   - `β = arctan(vr / sqrt(ur^2 + wr^2))`

【物理逻辑链】
风 → 空速变化 → 攻角/侧滑角变化 → 气动力/力矩变化 → 运动响应变化。

【图 4.9 含义】
白噪声经 Dryden 生成随机风 → 稳定风旋转到机体系 → 合成总风 → 计算空速 → 计算 α/β → 进入气动力模型。

【工程意义】
通过该链路可系统性评估“风对轨迹与姿态”的扰动效应，并指导飞控抗扰设计。
