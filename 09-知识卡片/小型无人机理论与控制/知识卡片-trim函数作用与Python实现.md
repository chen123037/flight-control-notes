### 主题：trim 函数作用与 Python 实现

【一句话定义】
trim 用数值求解平衡状态方程，给定目标飞行条件，求出稳定飞行所需的状态与控制输入。

【适用场景】
稳态飞行点计算、线性化建模基准点获取、仿真初始条件设置。

【核心概念】
1. **输入**：目标飞行条件（$V_a^\*$、$\gamma^\*$、$R^\*$）。
2. **输出**：平衡状态 $x^\*$ 与控制输入 $u^\*$。
3. **本质**：解非线性方程组 $F(x)=0$。

【关键公式/逻辑】
- 定义残差函数：$F(x)=0$（平衡条件/约束方程）
- 数值求解：`scipy.optimize.root` 或 `fsolve`
- 求解流程：残差定义 → 求根 → 还原完整状态与输入

【常见误区】
- 初始猜测差导致不收敛，需结合物理意义给初值。
- 只求状态不求控制，导致“平衡”不可实现。

【最小示例】
Python 伪流程：定义残差函数 → 调用 `root` 求解 → 计算 $u^\*$ 与机体系速度。
